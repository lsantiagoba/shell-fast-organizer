# Shell Fast Organizer

Add a right-click menu to the App Grid and Dash to automatically organize applications into category folders. Allows you to "Auto-organize All" or explicitly organize applications from one category (eg. Music, Video, Office).

## Features
- **Auto-organize All**: Scan all installed applications and group them into their corresponding categorical folders.
- **Organize by Category**: Organize existing applications by a specific individual category via the context menu.

## Compatibility
- GNOME Shell 45, 46, 47, 48, 49

## Installation
Run the provided `install.sh` script to install the extension to your local user GNOME Shell extensions folder and restart GNOME Shell.
